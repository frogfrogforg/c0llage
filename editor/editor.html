thanks for using forest editor v0.0001

very quick guide to making a page:

1. place the background image that you want to use in gatherings/forest/images

2. Click 'choose file' under 'choose an image'

3. Use the 'rectangle'/'circle'/'polygon' buttons to add hotspots

4. Press the 'edit' button to go into edit mode, where you can move and edit hotspot shapes

5. In edit mode you can double click a hotspot to change the href (link) and other attributes

6. Press 'save' to export as a .p.html file, which you should place in gatherings/forest

You can also load existing .p.html files to edit them, using the other 'choose file' button at the bottom. (Right now they have to be files that were exported from this editor.)

More detailed instructions below

Main

F5 — reload the page and display the form for loading image again

S — save map params in localStorage

Drawing mode (rectangle / circle / polygon)

ENTER — stop polygon drawing (or click on first helper)

ESC — cancel drawing of a new area

SHIFT — square drawing in case of a rectangle and right angle drawing in case of a polygon

Editing mode

DELETE — remove a selected area

ESC — cancel editing of a selected area

SHIFT — edit and save proportions for rectangle

I — edit attributes of a selected area (or dblclick on an area)

D — duplicate of the selected area

↑ — move a selected area up

↓ — move a selected area down

← — move a selected area to the left

→ — move a selected area to the right
